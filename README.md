# personal-financial-manager

A Java console-based application that helps users manage personal finances by tracking user income and expenses.

## Features
- Add income transactions
- Add expense transactions
- View account balance
- View income category totals
- View expense category totals
- Save transaction history to a file
- Exception handling for invalid input

## Technologies Used
- Java
- Object-Oriented Programming
- File Handling
- ArrayLists
- Exception Handling
